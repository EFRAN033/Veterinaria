En local (Windows), usar el Python del venv (el python global no tiene dependencias):
  cd back_veterinaria
  python -m venv .venv
  .\.venv\Scripts\pip install -r requirements.txt
  .\.venv\Scripts\python seed_users.py
  (necesitas archivo .env con DATABASE_URL y SECRET_KEY como en app/core/config)
